购买电动自行车攻略

发表于 2024-03-25 分类于 生活
本文字数： 2.4k 阅读时长 ≈ 2 分钟

确定需求和预算

个人需求

考虑你的骑行习惯、路线条件（城市通勤、越野等）、电动自行车的续航里程、速度、载重、折叠功能等。

  • 用于城市通勤，活动范围5km~15km
  • 续航里程不低于30km，载重两人。
  • 不需要折叠。
  • 最好有APP可以实时监控。

预算范围

预算2000~4000。

选购要点

选择电机

电机特性

电机是电动车关键部件，多数电动车使用低噪音、长寿命的无刷电机。

品牌与技术

主流电机品牌

  • 九州电机
  • 金宇星电机
  • 博世电机（Bosch）：德国品牌，以其高品质和技术创新而闻名。
  • 全顺电机
  • 安乃达电机
  • 星伟电机
  • 旺德福电机

电动车厂商自有电机品牌

  • 雅迪电机
  • 绿源电机
  • 爱玛电机

爱玛电动车部分型号配备进口无刷博世电机，属于高性能电机品牌之一。

功率与法规

虽然电机功率大可提升动力和速度，但2019年新国标限制电动车最高速度为25km/h，电机功率不超过400W。

电动车电机选择建议

  • 选择电机时，应考虑电机的功率、效率、耐用性和品牌声誉。
  • 高品质电机通常具有更好的性能、更长的使用寿命和更低的维护成本。
  • 对于电动车的使用需求和预算，选择合适的电机品牌非常重要。

选择电池

电池类型及其优缺点

单从安全方面来看，铅酸电池是最好的，其次锂电池中的磷酸铁锂＞锰酸锂＞三元锂

铅酸电池

  • 优点：

    1. 成本低廉：相比其他类型的电池，铅酸电池的成本较低。
    2. 技术成熟：铅酸电池技术历史悠久，制造和回收过程相对成熟。
    3. 维护简单：对于某些类型的铅酸电池，维护相对简单。
    4. 高可靠性：在适当的使用和维护条件下，铅酸电池可以非常可靠。

  • 缺点：

    1. 重量大：铅酸电池的能量密度较低，因此相对较重。
    2. 容量有限：相较锂电池，其储能容量较小。
    3. 充电速度慢：充电效率不如锂电池，需要较长时间充满。
    4. 环境污染：含铅成分对环境有害，需要谨慎处理。
    5. 使用寿命较短：使用寿命通常不如锂电池长久。
锂电池

  • 优点：

    1. 能量密度高：锂电池重量轻且容量大，适合需要轻便和大容量的应用。
    2. 充电速度快：相比铅酸电池，锂电池的充电效率更高。
    3. 使用寿命长：循环寿命通常比铅酸电池长。
    4. 自放电率低：在不使用的情况下，电量流失较慢。

  • 缺点：

    1. 成本较高：相比铅酸电池，锂电池的成本更高。
    2. 安全性问题：在某些情况下可能存在过热或燃烧的风险。
    3. 环境影响：虽然比铅酸电池环保，但锂电池回收利用仍然是挑战。
石墨烯电池

  • 优点：

    1. 充电速度极快：石墨烯电池的充电速度可以非常快。
    2. 寿命长：石墨烯电池的循环寿命可能比传统锂电池还要长。
    3. 能量密度高：理论上，石墨烯电池的能量密度超过传统锂电池。
    4. 热稳定性好：石墨烯电池在高温下的性能比传统电池更稳定。

  • 缺点：

    1. 成本问题：目前石墨烯电池的生产成本仍然较高。
    2. 技术成熟度：石墨烯电池技术相对较新，尚未大规模商业化。
    3. 生产挑战：大规模生产石墨烯电池面临技术和成本上的挑战。

选择控制器

电动车控制器10大品牌，电动车控制器排名前十名

电动车控制器的作用

电动车的控制器是核心部件，负责指挥车辆的前进、后退等操作，相当于人体的大脑。

控制器的规格

控制器按管数分类，有6管、9管、12管等不同规格。管数越多，输出功率越大，动力越强，但相应的耗电量也会增加。

散热性能

控制器的散热性能是关键指标，通常通过散热器的重量来判断，重量越重，散热能力越强。

选择轮胎

内外胎特点

  • 内外胎类似自行车轮胎，耐磨但容易爆胎且漏气快。
  • 爆胎后需立即寻找维修店进行补胎。

真空胎优势

  • 电动摩托车普遍采用真空胎，因其耐磨性好，漏气慢。
  • 补胎液使用寿命长，推荐选择真空胎以避免推车的尴尬。
  • 真空胎包括钢丝轮胎和芳纶防扎轮胎，提供更强的耐磨性和抗扎能力。

轮胎品牌推荐

  • 电动车轮胎多为定制采购。
  • 优质品牌包括朝阳、正新、建大、樱花。

选择制动系统

刹车系统类型及区别

  • 多数电动车采用前碟后鼓的制动组合。
  • 前轮在刹车时承担较大制动力，因此使用灵敏度高的碟刹。
  • 后轮承担较小的制动力且抓地力较低，使用鼓刹可以避免过度制动。
  • 前后轮均用碟刹并不会提升制动性能，反而可能降低稳定性并增加事故风险。
碟刹（Disc Brake）

碟刹采用金属制成的圆形刹车盘（碟片）和刹车卡钳。
当刹车时，卡钳中的刹车片夹紧碟片，产生摩擦力以减速。
碟刹散热性能好，即使在湿滑条件下也能保持较好的刹车效果。
通常重量较轻，维护和更换刹车片相对容易。

毂刹（Hub Brake）

毂刹是一种较老式的刹车系统，通常指的是鼓刹。
刹车时，刹车片向外扩张与刹车鼓内壁摩擦，产生制动力。
散热性能较差，重复或长时间刹车可能导致刹车效能下降。

鼓刹（Drum Brake）

鼓刹由刹车鼓和刹车片（刹车鞋）组成。
刹车时刹车片膨胀与鼓内壁接触，通过摩擦力减速。
与碟刹相比，鼓刹的散热性差，容易产生热衰退现象。
鼓刹结构封闭，对环境的抵抗力强，但维护和更换较为复杂。

区别总结

碟刹提供更好的散热和刹车性能，尤其在恶劣天气条件下。
鼓刹和毂刹结构相对封闭，适应性强，但散热性和刹车性能相对较弱。
碟刹维护简单，而鼓刹维护更加复杂。

选择续航能力

新国标电动车续航变化原因

新国标与旧国标电池差异

  1. 新国标电池材料减少，使电池更轻，影响续航。
  2. 新国标车型轮胎和电机功率增大，导致更高的电耗。
  3. 镉金属的禁用减少了电池耐温能力，但减少了环境污染。

冬季续航能力下降

  • 低温导致电池物质活性降低，续航能力减少。
  • 温度每降低15度，电池存储电量可减少约20%。
  • 冬季续航能力减少至约三分之一是正常现象。

锂电池冬季特性

  • 锂电池容量随温度变化而变化。
  • 低温下电导率下降，充电速度减慢，内阻增大可能导致过热。
  • BMS系统可能在低温下禁止充电，需在常温室内充电。

续航与电池容量

  • 选择大容量电池的电动车可提高续航，如12Ah至32Ah。
  • 续航能力较长的电动车价格较高，尤其是锂电池车型。

需求描述

家具的位置和功能

  1. 主卧储物柜定制
    • 主卧柜子：定制衣柜，考虑内部分隔和挂衣空间的合理布局，以适应不同衣物的收纳需求。
    • 门后柜子：可设计为开放式挂衣架或带储物功能的全身镜柜，利用空间同时提供便捷使用。
    • 飘窗上柜子：定制低柜或抽屉柜，与飘窗高度协调，提供额外储物空间，同时不阻挡窗户视线。
  2. 次卧储物柜定制
    • 根据房间大小和使用需求定制衣柜，可考虑结合书架或工作台，多功能使用。
  3. 书房储物柜和书桌定制
    • 书房柜子：定制书架和文件柜，以存放书籍和文档，考虑开放式和封闭式结合，以适应不同收纳需求。
    • 书桌：根据使用者习惯定制书桌大小和高度，考虑足够的腿部空间和电线管理功能。
  4. 客厅储物柜定制
    • 根据客厅布局定制电视柜、装饰柜或展示柜，考虑与整体家居风格的协调性和实用性。
  5. 餐厅储物柜定制
    • 定制餐边柜或酒柜，用于存放餐具、酒具和餐巾纸等，考虑易于取用的设计。
  6. 阳台储物柜定制
    • 定制耐候性强的储物柜，用于存放花卉园艺用品或清洁工具，考虑防水和防晒的特性。

家具中的甲醛

甲醛概述

  1. 定义与来源
    • 甲醛是一种无色有刺激性气味的化学物质，广泛用于建筑材料和家具制造中，尤其是在粘合剂或胶合板中。
    • 常见的释放甲醛的家具包括柜子、书架、床架等，特别是那些使用了大量人造板材的家具。
  2. 健康影响
    • 甲醛对人体有害，短期暴露可能引起眼睛、鼻子和喉咙的刺激，头痛和皮肤过敏。
    • 长期暴露可能增加患呼吸道疾病和癌症的风险。

家具中的甲醛问题

  1. 甲醛释放原因
    • 家具中使用的人造板材如刨花板、中密度纤维板（MDF）和胶合板，通常含有甲醛作为粘合剂的成分。
    • 随着时间的推移，这些材料会释放甲醛气体到室内环境中。
  2. 甲醛释放周期
    • 甲醛的释放通常在新家具安装后的头几年最为严重，尤其是在前六个月到一年内。
    • 释放速率会随着时间逐渐减慢，但在某些条件下可能会持续数年。

减少甲醛危害的措施

  1. 选择低甲醛产品
    • 在定制柜子时，选择标有“低甲醛”或“无甲醛”标签的材料。
    • 优先选用E0级或E1级的板材，这些级别的板材甲醛释放量较低。
  2. 保持室内通风
    • 在柜子安装后，确保房间保持良好通风，以帮助甲醛散发出去。
    • 尽可能经常开窗，特别是在最初的几周内。
  3. 使用甲醛清除剂
    • 可以购买市面上的甲醛清除剂，并按照说明书使用。
    • 一些清除剂可以直接喷洒在柜子表面，有的则可以放置在柜子内部吸附甲醛。
  4. 应用活性炭包
    • 活性炭具有很强的吸附能力，可以有效吸收甲醛等有害气体。
    • 放置活性炭包在柜子内部和房间的各个角落。
  5. 室内植物净化
    • 一些室内植物如绿萝、吊兰等被认为可以吸收甲醛。
    • 尽管植物的净化效果有限，但它们可以作为辅助手段，改善室内空气质量。
  6. 使用专业检测与治理服务
    • 如果担心甲醛超标，可以请专业的室内环境检测公司进行检测。
    • 根据检测结果，如果甲醛超标，可以请专业公司进行治理。
  7. 注意日常护理
    • 避免使用含有甲醛的清洁剂和涂料。
    • 定期清洁柜子，防止灰尘积聚，因为灰尘可能吸附甲醛。
  8. 等待自然挥发
    • 甲醛会随着时间的推移而逐渐挥发，通常在安装新家具后的6-12个月内甲醛含量会降低。
    • 如果不是特别敏感，可以选择在此期间减少使用新柜子的频率，等待甲醛自然降低。

购买家具询问问题

  1. 材质和工艺
    • 问题内容：询问家具所使用的材料类型（如实木、板材、金属、玻璃等），以及相关的工艺标准（如木材干燥程度、油漆类型等）。
  2. 安全和环保标准
    • 问题内容：了解家具是否符合安全标准，是否使用了无害或低毒的材料，是否有相关的环保认证。
  3. 设计和尺寸

技术原理

原理讲解

虚拟滚动的实现步骤：

  1. 当内容区域大于可视区域时，可视区域可以滚动
  2. 当滚动时，计算内容区域，具体步骤如下：
    1. 计算本次滚动的距离
    2. 根据滚动距离计算移动了多少个内容
    3. 内容列表从头部删除移动的内容，同时内容的上间距增加删除内容的高度
    4. 内容列表底部增加删除个数的内容
  3. 滚动过程中不断重复第2步即可

购房预算

毛坯房：150W以内

精装修：160W以内

首付：50W以内

每月分期：6000以内

地段需求

  • 尽量三环内，最多不超过四环
  • 距离东站地铁或公交通勤在1小时内，尽量不超过1小时20分钟
  • 海拔尽量要大于90米（高新>中原>管城>=金水>经开）
  • 不能离高科技产业和金融产业太远

环境需求

  • 不能太吵闹，小区附近不能有高速、火车、机场，尽量不要临街
  • 交通要方便，尽量离地铁近
  • 购物要方便，附近要有超市

  • 房子建成要在2年到8年

配套需求

如何规划看房计划

  1. 筛选出合适的中介
  2. 根据自己看中的小区和中介推荐的小区规划行程
  3. 与中介约定看房的时间和地点

看房的要点有哪些

  1. 看小区周围是否影响环境或者健康的设施
    1. 小区附近是否存在墓地、垃圾场、高压线、电厂、化工厂、高速路
  2. 查看小区的配套
    1. 搜索小区周边的地铁、超市、医院、学校、公园。
  3. 查看小区环境
    1. 小区安保是否完善
    2. 小区是否整洁
    3. 小区的绿化率
    4. 小区停车位是否充足
  4. 查看大楼环境
    1. 大楼之间的间距
    2. 大楼的高度
    3. 大楼外层是否有脱落、裂缝、漏水
    4. 楼里过道是否有占用
    5. 电梯是否老旧
    6. 大楼是否临街
  5. 查看房屋环境
    1. 户型是否是合理、动静分离、所有房间都明亮
    2. 检查是否存在漏水、房屋的墙角、厨房底部水槽、洗手间的地板和四周、阳台顶部墙角和地板
    3. 查看承重墙、主梁、地板是否有裂纹。（1.通过声音判断，敲击墙体，没有太多的声音为承重墙，有清脆的大回声的是非承重墙。2.通过厚度判断，非承重墙的墙体厚度明显比承重墙薄。3.通过部位判断，外墙和邻居共用的墙通常都是承重墙，卫生间、储藏间的墙一般都是非承重墙。）
    4. 检查门窗是否存在毁坏或者故障
    5. 检查插头、电器是否可以使用
  6. 询问房屋信息
    1. 房屋满几年，有哪些税费
    2. 是否有抵押、租赁
    3. 是否满五唯一
    4. 房子的产权人有哪些
    5. 房屋的户籍情况，是否有非产权人在户口下，尤其是否有学区名额
