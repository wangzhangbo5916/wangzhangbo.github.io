购买电动自行车攻略

发表于 2024-03-25 更新于 2024-03-26 分类于 生活 阅读次数：
本文字数： 2.4k 阅读时长 ≈ 2 分钟

确定需求和预算

个人需求

考虑你的骑行习惯、路线条件（城市通勤、越野等）、电动自行车的续航里程、速度、载重、折叠功能等。

  • 用于城市通勤，活动范围5km~15km
  • 续航里程不低于30km，载重两人。
  • 不需要折叠。
  • 最好有APP可以实时监控。

预算范围

预算2000~4000。

选购要点

选择电机

电机特性

电机是电动车关键部件，多数电动车使用低噪音、长寿命的无刷电机。

品牌与技术

主流电机品牌

  • 九州电机
  • 金宇星电机
  • 博世电机（Bosch）：德国品牌，以其高品质和技术创新而闻名。
  • 全顺电机
  • 安乃达电机
  • 星伟电机
  • 旺德福电机

电动车厂商自有电机品牌

  • 雅迪电机
  • 绿源电机
  • 爱玛电机

爱玛电动车部分型号配备进口无刷博世电机，属于高性能电机品牌之一。

功率与法规

虽然电机功率大可提升动力和速度，但2019年新国标限制电动车最高速度为25km/h，电机功率不超过400W。

电动车电机选择建议

  • 选择电机时，应考虑电机的功率、效率、耐用性和品牌声誉。
  • 高品质电机通常具有更好的性能、更长的使用寿命和更低的维护成本。
  • 对于电动车的使用需求和预算，选择合适的电机品牌非常重要。

选择电池

电池类型及其优缺点

单从安全方面来看，铅酸电池是最好的，其次锂电池中的磷酸铁锂＞锰酸锂＞三元锂

铅酸电池

  • 优点：

    1. 成本低廉：相比其他类型的电池，铅酸电池的成本较低。
    2. 技术成熟：铅酸电池技术历史悠久，制造和回收过程相对成熟。
    3. 维护简单：对于某些类型的铅酸电池，维护相对简单。
    4. 高可靠性：在适当的使用和维护条件下，铅酸电池可以非常可靠。

  • 缺点：

    1. 重量大：铅酸电池的能量密度较低，因此相对较重。
    2. 容量有限：相较锂电池，其储能容量较小。
    3. 充电速度慢：充电效率不如锂电池，需要较长时间充满。
    4. 环境污染：含铅成分对环境有害，需要谨慎处理。
    5. 使用寿命较短：使用寿命通常不如锂电池长久。
锂电池

  • 优点：

    1. 能量密度高：锂电池重量轻且容量大，适合需要轻便和大容量的应用。
    2. 充电速度快：相比铅酸电池，锂电池的充电效率更高。
    3. 使用寿命长：循环寿命通常比铅酸电池长。
    4. 自放电率低：在不使用的情况下，电量流失较慢。

  • 缺点：

    1. 成本较高：相比铅酸电池，锂电池的成本更高。
    2. 安全性问题：在某些情况下可能存在过热或燃烧的风险。
    3. 环境影响：虽然比铅酸电池环保，但锂电池回收利用仍然是挑战。
石墨烯电池

  • 优点：

    1. 充电速度极快：石墨烯电池的充电速度可以非常快。
    2. 寿命长：石墨烯电池的循环寿命可能比传统锂电池还要长。
    3. 能量密度高：理论上，石墨烯电池的能量密度超过传统锂电池。
    4. 热稳定性好：石墨烯电池在高温下的性能比传统电池更稳定。

  • 缺点：

    1. 成本问题：目前石墨烯电池的生产成本仍然较高。
    2. 技术成熟度：石墨烯电池技术相对较新，尚未大规模商业化。
    3. 生产挑战：大规模生产石墨烯电池面临技术和成本上的挑战。

选择控制器

电动车控制器10大品牌，电动车控制器排名前十名

电动车控制器的作用

电动车的控制器是核心部件，负责指挥车辆的前进、后退等操作，相当于人体的大脑。

控制器的规格

控制器按管数分类，有6管、9管、12管等不同规格。管数越多，输出功率越大，动力越强，但相应的耗电量也会增加。

散热性能

控制器的散热性能是关键指标，通常通过散热器的重量来判断，重量越重，散热能力越强。

选择轮胎

内外胎特点

  • 内外胎类似自行车轮胎，耐磨但容易爆胎且漏气快。
  • 爆胎后需立即寻找维修店进行补胎。

真空胎优势

  • 电动摩托车普遍采用真空胎，因其耐磨性好，漏气慢。
  • 补胎液使用寿命长，推荐选择真空胎以避免推车的尴尬。
  • 真空胎包括钢丝轮胎和芳纶防扎轮胎，提供更强的耐磨性和抗扎能力。

轮胎品牌推荐

  • 电动车轮胎多为定制采购。
  • 优质品牌包括朝阳、正新、建大、樱花。

选择制动系统

刹车系统类型及区别

  • 多数电动车采用前碟后鼓的制动组合。
  • 前轮在刹车时承担较大制动力，因此使用灵敏度高的碟刹。
  • 后轮承担较小的制动力且抓地力较低，使用鼓刹可以避免过度制动。
  • 前后轮均用碟刹并不会提升制动性能，反而可能降低稳定性并增加事故风险。
碟刹（Disc Brake）

碟刹采用金属制成的圆形刹车盘（碟片）和刹车卡钳。
当刹车时，卡钳中的刹车片夹紧碟片，产生摩擦力以减速。
碟刹散热性能好，即使在湿滑条件下也能保持较好的刹车效果。
通常重量较轻，维护和更换刹车片相对容易。

毂刹（Hub Brake）

毂刹是一种较老式的刹车系统，通常指的是鼓刹。
刹车时，刹车片向外扩张与刹车鼓内壁摩擦，产生制动力。
散热性能较差，重复或长时间刹车可能导致刹车效能下降。

鼓刹（Drum Brake）

鼓刹由刹车鼓和刹车片（刹车鞋）组成。
刹车时刹车片膨胀与鼓内壁接触，通过摩擦力减速。
与碟刹相比，鼓刹的散热性差，容易产生热衰退现象。
鼓刹结构封闭，对环境的抵抗力强，但维护和更换较为复杂。

区别总结

碟刹提供更好的散热和刹车性能，尤其在恶劣天气条件下。
鼓刹和毂刹结构相对封闭，适应性强，但散热性和刹车性能相对较弱。
碟刹维护简单，而鼓刹维护更加复杂。

选择续航能力

新国标与旧国标电池差异

  1. 新国标电池材料减少，使电池更轻，影响续航。
  2. 新国标车型轮胎和电机功率增大，导致更高的电耗。
  3. 镉金属的禁用减少了电池耐温能力，但减少了环境污染。

冬季续航能力下降

  • 低温导致电池物质活性降低，续航能力减少。
  • 温度每降低15度，电池存储电量可减少约20%。
  • 冬季续航能力减少至约三分之一是正常现象。

锂电池冬季特性

  • 锂电池容量随温度变化而变化。
  • 低温下电导率下降，充电速度减慢，内阻增大可能导致过热。
  • BMS系统可能在低温下禁止充电，需在常温室内充电。

续航与电池容量

  • 选择大容量电池的电动车可提高续航，如12Ah至32Ah。
  • 续航能力较长的电动车价格较高，尤其是锂电池车型。
