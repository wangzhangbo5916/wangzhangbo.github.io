买房攻略~选房流程

发表于 2023-08-30 更新于 2023-09-27 分类于 生活 阅读次数：
本文字数： 5.7k 阅读时长 ≈ 5 分钟

描述需求

购房预算

毛坯房：150W以内

精装修：160W以内

首付：50W以内

每月分期：6000以内

地段需求

  • 尽量三环内，最多不超过四环
  • 距离东站地铁或公交通勤在1小时内，尽量不超过1小时20分钟
  • 海拔尽量要大于90米（高新>中原>管城>=金水>经开）
  • 不能离高科技产业和金融产业太远

环境需求

  • 不能太吵闹，小区附近不能有高速、火车、机场，尽量不要临街
  • 交通要方便，尽量离地铁近
  • 购物要方便，附近要有超市

房子需求

  • 房子所在的楼尽量不要临街
  • 房子建成要在2年到8年
  • 房屋套内使用面积不小于75平方米
  • 房子尽量要双卫
  • 房子所在的楼层高尽量不要超过20层
  • 房子所在的楼层的四层以上，最多不超过2/3
  • 房子不能出现过大型火灾事故以及人员死亡
  • 房子的产权必须清晰，无抵押，无租赁
  • 房子尽量南北通透和动静分离

配套需求

  • 开发商尽量是国企央企
  • 物业尽量要比较有口碑的
  • 车位要比较充足

筛选房子

根据购房预算和区域房价筛选区域

金水区、中原区、管城回族区、二七区、郑东新区

看房的要点有哪些

  1. 看小区周围是否影响环境或者健康的设施
    1. 小区附近是否存在墓地、垃圾场、高压线、电厂、化工厂、高速路
  2. 查看小区的配套
    1. 搜索小区周边的地铁、超市、医院、学校、公园。
  3. 查看小区环境
    1. 小区安保是否完善
    2. 小区是否整洁
    3. 小区的绿化率
    4. 小区停车位是否充足
  4. 查看大楼环境
    1. 大楼之间的间距
    2. 大楼的高度
    3. 大楼外层是否有脱落、裂缝、漏水
    4. 楼里过道是否有占用
    5. 电梯是否老旧
    6. 大楼是否临街
  5. 查看房屋环境
    1. 户型是否是合理、动静分离、所有房间都明亮
    2. 检查是否存在漏水、房屋的墙角、厨房底部水槽、洗手间的地板和四周、阳台顶部墙角和地板
    3. 查看承重墙、主梁、地板是否有裂纹。（1.通过声音判断，敲击墙体，没有太多的声音为承重墙，有清脆的大回声的是非承重墙。2.通过厚度判断，非承重墙的墙体厚度明显比承重墙薄。3.通过部位判断，外墙和邻居共用的墙通常都是承重墙，卫生间、储藏间的墙一般都是非承重墙。）
    4. 检查门窗是否存在毁坏或者故障
    5. 检查插头、电器是否可以使用
  6. 询问房屋信息
    1. 房屋满几年，有哪些税费
    2. 是否有抵押、租赁
    3. 是否满五唯一
    4. 房子的产权人有哪些
    5. 房屋的户籍情况，是否有非产权人在户口下，尤其是否有学区名额
